## 📦 构建部署

### 前端构建
```bash
cd web-vue
npm run build
# 输出到 dist/
```

### 集成到Go
```bash
# 复制前端构建产物
cp -r web-vue/dist/* web/static/

# 构建Go
go build -o lazy-auto-ops cmd/server/main.go
```

### Docker部署
```dockerfile
# 多阶段构建
FROM node:18 AS frontend
WORKDIR /app/web-vue
COPY web-vue/package*.json ./
RUN npm install
COPY web-vue/ ./
RUN npm run build

FROM golang:1.21 AS backend
WORKDIR /app
COPY go.mod go.sum ./
RUN go mod download
COPY . .
COPY --from=frontend /app/web-vue/dist ./web/static
RUN go build -o lazy-auto-ops cmd/server/main.go

FROM debian:bullseye-slim
WORKDIR /app
COPY --from=backend /app/lazy-auto-ops .
COPY --from=backend /app/configs ./configs
COPY --from=backend /app/web ./web
EXPOSE 8080
CMD ["./lazy-auto-ops"]
```
