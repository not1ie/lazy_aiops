## 🎯 新增功能

### 1. 实时监控数据采集器 (完成度: 85% → 95%)

#### 核心功能
- ✅ **系统指标采集**
  - CPU使用率和负载
  - 内存使用情况
  - 磁盘使用情况
  - 网络流量统计
  - 主机状态监控

- ✅ **采集器特性**
  - 定时自动采集（可配置间隔）
  - 并发采集提高效率
  - 跨平台支持（Linux/macOS/Windows）
  - 数据持久化存储
  - 自动清理历史数据

- ✅ **实时数据访问**
  - 获取当前实时指标
  - 查询历史监控数据
  - 支持时间范围筛选

#### 数据模型
```go
type SystemMetrics struct {
    Timestamp   time.Time
    CPU         CPUMetrics      // CPU指标
    Memory      MemoryMetrics   // 内存指标
    Disk        DiskMetrics     // 磁盘指标
    Network     NetworkMetrics  // 网络指标
    Hosts       []HostMetrics   // 主机指标
}

type MetricRecord struct {
    Timestamp   time.Time
    CPUUsage    float64
    MemoryUsage float64
    DiskUsage   float64
    NetworkIn   uint64
    NetworkOut  uint64
}
```

#### 新增API
```
GET /api/v1/monitor/metrics/realtime   # 获取实时监控指标（完整）
GET /api/v1/monitor/metrics/history    # 获取历史监控数据
```

#### 配置示例
```yaml
plugins:
  monitor:
    enabled: true
    config:
      interval: 30  # 采集间隔（秒），默认30秒
```

### 2. RBAC权限管理系统 (完成度: 0% → 100%)

#### 核心功能
- ✅ **用户管理**
  - 用户CRUD操作
  - 密码管理（加密存储）
  - 用户状态管理（启用/禁用）
  - 用户角色分配

- ✅ **角色管理**
  - 角色CRUD操作
  - 角色权限配置
  - 角色保护（admin角色不可删除）

- ✅ **权限管理**
  - 权限CRUD操作
  - 权限树形结构
  - 权限类型（menu/button/api）
  - 默认权限初始化

- ✅ **操作日志**
  - 自动记录所有操作
  - 日志查询和筛选
  - 操作审计追踪

#### 数据模型
```go
type User struct {
    Username string
    Password string  // bcrypt加密
    Nickname string
    Email    string
    Phone    string
    Status   int     // 1:启用 0:禁用
    RoleID   string
    Role     *Role
}

type Role struct {
    Name        string
    Code        string
    Description string
    Permissions []*Permission
}

type Permission struct {
    Name     string
    Code     string  // 如: cmdb:host:read
    Type     string  // menu, button, api
    ParentID string
}

type OperationLog struct {
    UserID    string
    Username  string
    Module    string
    Action    string
    Target    string
    Detail    string
    IP        string
    UserAgent string
    Status    int
}
```

#### API接口
```
# 用户管理
GET    /api/v1/rbac/users              # 用户列表
POST   /api/v1/rbac/users              # 创建用户
GET    /api/v1/rbac/users/:id          # 用户详情
PUT    /api/v1/rbac/users/:id          # 更新用户
DELETE /api/v1/rbac/users/:id          # 删除用户
PUT    /api/v1/rbac/users/:id/password # 修改密码
PUT    /api/v1/rbac/users/:id/status   # 修改状态

# 角色管理
GET    /api/v1/rbac/roles              # 角色列表
POST   /api/v1/rbac/roles              # 创建角色
GET    /api/v1/rbac/roles/:id          # 角色详情
PUT    /api/v1/rbac/roles/:id          # 更新角色
DELETE /api/v1/rbac/roles/:id          # 删除角色
PUT    /api/v1/rbac/roles/:id/permissions # 更新角色权限

# 权限管理
GET    /api/v1/rbac/permissions        # 权限列表
POST   /api/v1/rbac/permissions        # 创建权限
GET    /api/v1/rbac/permissions/:id    # 权限详情
PUT    /api/v1/rbac/permissions/:id    # 更新权限
DELETE /api/v1/rbac/permissions/:id    # 删除权限
GET    /api/v1/rbac/permissions/tree   # 权限树

# 操作日志
GET    /api/v1/rbac/logs               # 操作日志列表
GET    /api/v1/rbac/logs/:id           # 日志详情
```

#### 默认权限
系统自动初始化以下权限：
- 仪表板（dashboard）
- 资产管理（cmdb）
- 监控中心（monitor）
- 告警管理（alert）
- 任务调度（task）
- AI分析（ai）
- 系统管理（system）
  - 用户管理
  - 角色管理
  - 权限管理
  - 操作日志

#### 使用示例

**1. 创建用户**
```bash
curl -X POST http://localhost:8080/api/v1/rbac/users \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{
    "username": "zhangsan",
    "password": "password123",
    "nickname": "张三",
    "email": "zhangsan@example.com",
    "role_id": "role-id-here"
  }'
```

**2. 创建角色**
```bash
curl -X POST http://localhost:8080/api/v1/rbac/roles \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{
    "name": "运维工程师",
    "code": "ops",
    "description": "负责日常运维工作"
  }'
```

**3. 分配权限**
```bash
curl -X PUT http://localhost:8080/api/v1/rbac/roles/{role_id}/permissions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{
    "permission_ids": [
      "perm-id-1",
      "perm-id-2",
      "perm-id-3"
    ]
  }'
```

**4. 修改密码**
```bash
curl -X PUT http://localhost:8080/api/v1/rbac/users/{user_id}/password \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{
    "old_password": "old123",
    "new_password": "new123"
  }'
```

**5. 查询操作日志**
```bash
# 查询所有日志
curl http://localhost:8080/api/v1/rbac/logs \
  -H "Authorization: Bearer <token>"

# 按用户筛选
curl "http://localhost:8080/api/v1/rbac/logs?username=admin" \
  -H "Authorization: Bearer <token>"

# 按模块筛选
curl "http://localhost:8080/api/v1/rbac/logs?module=user" \
  -H "Authorization: Bearer <token>"
```

#### 安全特性
1. **密码安全**
   - bcrypt加密存储
   - 密码强度验证（可扩展）
   - 密码修改需验证旧密码

2. **权限控制**
   - 基于角色的访问控制（RBAC）
   - 细粒度权限管理
   - 权限继承和组合

3. **操作审计**
   - 所有操作自动记录
   - 包含用户、时间、IP、操作详情
   - 支持审计追踪

4. **保护机制**
   - admin用户不可删除
   - admin角色不可删除
   - 角色下有用户时不可删除
   - 用户状态控制
